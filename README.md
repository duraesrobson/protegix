# PROTEGIX - Segurança Digital

Projeto desenvolvido com foco em conscientização sobre **segurança digital**, permitindo a visualização de dados reais coletados via Google Forms, além de oferecer um quiz educativo e análises automatizadas.

---

## Funcionalidades

### Dashboard de Dados
- Exibe respostas do Google Forms em gráficos interativos
- Atualização dinâmica via Google Sheets
- Visualização simples e acessível

### Análise com IA
- Geração de insights automáticos sobre os dados coletados
- Estrutura preparada para integração com IA gratuita (Gemini / Hugging Face)

### Quiz Interativo
- Perguntas sobre segurança digital
- Feedback imediato (correto = verde / errado = vermelho)
- Explicação após cada resposta
- Navegação entre perguntas

---

## Tecnologias Utilizadas

- **React**
- **TypeScript**
- **Vite**
- **Recharts** (gráficos)
- **Axios** (requisições)
- **Google Forms + Google Sheets**
- **Vercel** (deploy)

## Objetivo

Promover a conscientização sobre segurança digital através de:
- dados reais
- visualização clara
- aprendizado interativo
